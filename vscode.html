vscode core shortkey vowel

  • ctrl+k+f 선택부분 자동 들여쓰기 및 스타일 포멧 (많이 쓰는 플러그인)
  • ctrl+b 사이드바 여닫기
  • alt+ 방향키 위 아래 한단씩 이동
  • shift+alt 방향키 위아래 한단 복사
  • ctrl+x 한 단 삭제
  • ctrl + w 탭 닫기

  • alt+커서 멀티 커서
  • ctrl+alt+up/down 세로 다중 커서
  • ctrl+shift+i 선택 줄 끝에 커서

  • ctrl+d 같은 단어 밑으로 선택
  • ctrl+shift+l 같은 단어 전체 선택
  • ctrl+h 모든 단어 바꾸기

  • ctrl+f 텍스트 찾기
  • ctrl+p 이름으로 파일 검색
  • ctrl + shift + p 모든 명령 검색

  • ctrl + z 되돌리기
  • ctrl + y 돌아오기
  • tab 들여쓰기
  • shift + tab 내어쓰기
  • ctrl + 숫자 : 창 나누기
  • ctrl + \ : 현재창 나누기
  • ctrl + pageup/pagedown : 창간 이동

  • 선택 후+ ( 자동으로 열고 닫아주기
  • ctrl +- 창 키우거나 줄이기
  • ctrl + / 주석넣기
